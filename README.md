# PipelineMips

В этом проекте реализован 32-битный конвейерный процессор MIPS с использованием SystemVerilog. Процессор построен по модульному принципу и состоит из стандартных пяти этапов конвейера: выборка инструкций (IF), декодирование инструкций (ID), выполнение (EX), обращение к памяти (MEM) и обратная запись (WB). Реализация была проверена с помощью относительно сложной тестовой программы - КИХ-фильтра скользящего среднего.

Структурная схема модуля верхнего уровня состоит состоящего из модулей памяти данных и инструкций, а также устройства управления и трактом данных, представляющим собой логику путешествия данных между функциональными блоками процессора. При этом принимается, что и память команд, и память данных находятся снаружи процессора и подсоединены к нему с помощью шин адреса и данных. Это более реалистичный сценарий, потому что большинство процессоров используют внешнюю память.  Оно содержит модульную структуру процессора и включает в себя пять конвейерных блоков и четыре конвейерных регистра. Для каждого компонента процессора в коде предусмотрен отдельный файл.

Конструкция поддерживает арифметические, логические операции, операции с памятью, ветвления и переходы.
